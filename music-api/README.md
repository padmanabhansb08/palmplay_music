# PalmPlay Music API

Self-hosted JioSaavn API for PalmPlay. Deployed on Render.

## Deploy on Render

1. Go to [render.com](https://render.com) → New → Web Service
2. Connect your GitHub repo: `padmanabhansb08/palmplay_music`
3. Settings:
   - **Name:** `palmplay-music-api`
   - **Root Directory:** `music-api`
   - **Runtime:** Docker
   - **Instance Type:** Free
4. Click **Deploy**

Once live, copy the URL (e.g. `https://palmplay-music-api.onrender.com`) and update `catalog-config.js`.
